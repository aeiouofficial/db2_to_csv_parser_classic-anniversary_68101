# db2_to_csv_parser_classic-anniversary_68101
Standalone **WoW Classic - Burning Crusade Anniversary (build 2.5.5.68101)** DB2 -> CSV exporter.  It connects to the Blizzard CASC CDN, pulls the shipped `*.db2` client database tables for build `68101`, decodes them, and writes them as CSV into `assets/db/68101/`. Fully self-contained -- no game repo required.
